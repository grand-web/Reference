Значения цвета

В HTML цвета также могут быть заданы с помощью значений RGB, шестнадцатеричных значений (hex), HSL значений, RGBA значений и HSLA значений:

RGB (red, green, blue)

#hexadecimal-color-value

HSL (hue, saturation, lightness) (оттенок, насыщенность, легкость)

RGBA (red, green, blue, alpha)

HSLA (hue, saturation, lightness, alpha)

Например, так же, как название цвет "Tomato":

rgb(255, 99, 71)
#ff6347
hsl(9, 100%, 64%)

Так же, как название цвета "Tomato", но 50% прозрачный:

rgba(255, 99, 71, 0.5)
hsla(9, 100%, 64%, 0.5)

Прозрачность

В CSS есть три способа изменить прозрачность элемента:
с помощью функции rgba(),
с помощью функции hsla(),
c помощью свойства opacity - ПРЕДПОЧТИТЕЛЬНО. Так как основной цвет в таком случае, может свободно выбираться с измением стиля.

Свойство opacity позволяет сделать любой элемент веб-страницы частично или полностью прозрачным. Данное свойство изменяет прозрачность элементов, для которых установлено фоновое изображение (картинка) или задан фон с помощью цвета или градиента. Если элемент, для которого применено свойство opacity, содержит внутри себя другие элементы, то они также изменят свою прозрачность.
Свойство принимает значения в диапазоне от 0 (полностью прозрачный) до 1 (непрозрачный) или проценты в пределах от 0% до 100%. Можно выставить несколько чисел после запятой, например "0.44". То же значение принимает и альфа-канал (rgba,hsla)
